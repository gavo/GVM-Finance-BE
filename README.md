# GVM-Finance-BE

Backend for my Finance App

## Requirements

- Java 21 LTS
- Maven 3.8.7
- Docker 29.0.3
- PostgreSQL 17.5

## API Documentation

Swagger UI is available at `http://localhost:8080/swagger-ui.html`

## How to Build and Run Locally

1. Clone the repo:

   ```bash
   git clone https://github.com/gavo/GVM-Finance-BE.git
   ```

2. Build the project:

   ```bash
   mvn clean install
   ```

3. Run with Docker:
   ```bash
   docker-compose up
   ```
